# Acknowledgements

W.O.L.K. is built on top of a stack of excellent open-source projects. This
file is a human-readable acknowledgement summary, not a complete legal bill of
materials. For exact package versions, see `package.json` and `yarn.lock`.

## Core Application Stack

- [Electron](https://www.electronjs.org/) for the desktop application shell
- [Electron Forge](https://www.electronforge.io/) for packaging and distribution
- [Vue 3](https://vuejs.org/) for the renderer UI
- [Vue Router](https://router.vuejs.org/) for in-app navigation
- [Vite](https://vitejs.dev/) for development and renderer bundling
- [TypeScript](https://www.typescriptlang.org/) for typed application code

## Editing, Rendering, And Motion

- [Three.js](https://threejs.org/) for 3D rendering
- [TipTap](https://tiptap.dev/) for rich-text editing foundations
- [Meyda](https://meyda.js.org/) for audio feature extraction
- [vuedraggable](https://github.com/SortableJS/vue.draggable.next) for drag-and-drop UI behavior

## Testing And Tooling

- [Vitest](https://vitest.dev/) for automated testing
- [Vue Test Utils](https://test-utils.vuejs.org/) for Vue component testing
- [ESLint](https://eslint.org/) for linting
- [Sass](https://sass-lang.com/) for stylesheet authoring

## Icons And Interface Assets

- [Material Design Icons](https://pictogrammers.com/library/mdi/) via `@mdi/js`
- [`@jamescoyle/vue-icon`](https://github.com/jamescoyle/vue-icon) for SVG icon rendering

## Archive And File Utilities

- [`yauzl`](https://github.com/thejoshwolfe/yauzl) for ZIP reading
- [`yazl`](https://github.com/thejoshwolfe/yazl) for ZIP writing

## Thanks

Thanks to the maintainers and contributors of the projects above. This app
would be much harder to build, test, and ship without their work.
